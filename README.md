# Clinic Appointment Booking System

## Features
- Patient appointment booking
- QR code based access
- Admin panel for doctor
- Approve / Reject appointments
- Email notifications using PHPMailer
- Duplicate slot prevention

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- PHPMailer
